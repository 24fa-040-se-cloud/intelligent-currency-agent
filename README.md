# 💱 Intelligent Currency Converter Agent

## 📌 Overview

This project is an AI-based currency converter that uses multiple search algorithms to find optimal conversion paths between currencies.

## 🧠 Features

* Autonomous agent decision making
* Multiple algorithms:

  * BFS
  * DFS
  * UCS
  * A* Search
* Algorithm comparison (time, nodes explored)
* GUI built with PyQt5
* Currency trend graph visualization

## ⚙️ How It Works

User Input → Agent Decision → Algorithm Execution → Result

The agent automatically selects the best algorithm:

* BFS for direct conversion
* A* for multi-step optimization

## 📊 Additional Features

* Direct vs optimized comparison
* Swap currencies
* 7-day trend graph

## 🛠️ Technologies Used

* Python
* PyQt5
* Matplotlib

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
```
